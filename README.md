Бот “Організатор подій”

Цей проєкт реалізує простий консольний застосунок на Python для керування подіями. Програма дозволяє користувачу створювати записи про майбутні події, переглядати їх у будь-який момент, а також змінювати або видаляти за потреби. Такий інструмент можна використовувати для планування навчальних занять, дедлайнів, зустрічей або особистих справ.

Уся інформація про події зберігається у файлі events.json. Під час запуску програми файл читається і дані завантажуються у список, з яким працює програма. Коли користувач додає нову подію або змінює вже існуючу, файл автоматично перезаписується, тому дані залишаються доступними при наступному запуску.

Кожен запис події містить ідентифікатор, назву, дату проведення, час початку та короткий опис або категорію. Така структура дозволяє легко знаходити потрібну подію, редагувати її або видаляти.

Програма запускається через файл bot.py. Після запуску користувач вводить команди у терміналі, а система обробляє введений текст і виконує відповідну дію з даними. Завдяки цьому застосунок демонструє роботу з файлами, обробку введення користувача та використання стандартних можливостей Python для роботи з датами і часом."# Project-1"
